# TowerBound
Proyecto usando Twine y Harlowe de historia interactiva para Informática Audiovisual.
Realizado por Ana Pérez Bango y Omar Aguirre Rodríguez.
<img src="https://github.com/AnaPB8/TowerBound/blob/main/img/logo_DEF.png" alt="Logo de TowerBound">
## Acompaña al mago y a Cubito de Hielo en sus aventuras para salir de la torre en la que se encuentran atrapados.
- 17 finales posibles dependiendo de las acciones a tomar.
- Puzles y retos interactivos durante el transcurso de la historia.
- Una sorpresa te espera si descubres todos los finales...
